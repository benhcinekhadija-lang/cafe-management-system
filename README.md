 Cafe Management System

A web-based cafe management system developed using HTML, CSS, and JavaScript.

# Features

- Product management (add, edit, delete products)
- Inventory management and low stock alerts
- Point of sale (POS) system
- Sales history tracking
- Invoice generation and printing
- Statistics and reports
- Data storage using LocalStorage

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Project Purpose

This project was developed to practice front-end web development and create a simple system for managing a cafe's daily operations.
